# Contacts Manager CLI

A command line application to manage your contacts built with Node.js.

## Features
- Add a new contact
- View all contacts
- Delete a contact by ID
- Update a contact by ID

## Tech Stack
- Node.js
- fs module

## How to Run
node index.js add "Name" "Phone" "Email"
node index.js view
node index.js delete <id>
node index.js update <id> "Name" "Phone" "Email"
